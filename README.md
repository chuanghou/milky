# milky

引入方式
```xml
<dependency>
    <groupId>com.stellariver.milky</groupId>
    <artifactId>milky-spring-boot-starter</artifactId>
    <version>latest</version>
</dependency>
```

            
提供bom文件
```xml
<dependency>
    <groupId>com.stellariver.milky</groupId>
    <artifactId>milky-dependencies</artifactId>
    <version>latest</version>
</dependency>
```
详细使用方式请看demo模块及相应测试
